Farming Turtle

Description

Automatic farming turtle for a square farm. Refuels efficiently to ensure it will complete a full loop of the farm. 
This code works with any single crop.

Setup

Place the turtle on top of a chest, at the bottom left corner of the farm. It needs to be on the bottom perimeter of the farm.

Place fuel in the first slot, and place the crop/seed you want planting in the second slot.

Then, install the script using this command:

pastebin get hKYdhSjm farm

Then simply run it with:

farm

It will ask for the size of the farm. This is the width/height of the farmland area. So with the example image above, it would be a size of 9.
